🚀 HOW TO START (Choose One):

  OPTION 1: Quick Start (Easiest)
  ─────────────────────────────────
  1. Find index.html in this folder
  2. Double-click it
  3. Browser opens → Enjoy! 🎉
  
  OPTION 2: Using Local Server (Recommended)
  ───────────────────────────────────────────
  1. Open PowerShell in this folder
  2. Run: python -m http.server 8000
  3. Open browser: http://localhost:8000
  4. Explore the app! 🎉

🆘 NEED HELP?

  Problem: Can't open the app
  Solution: Use a local server (see QUICKSTART.md)
  
  Problem: Login doesn't work
  Solution: Check browser console (F12) for errors
  
  Problem: Page looks broken
  Solution: Clear cache (Ctrl+Shift+Delete) and hard refresh
  
  Problem: Lost session after refresh
  Solution: Check "Remember Me" during login
  
  See README.md for more troubleshooting!
